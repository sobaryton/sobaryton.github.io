Front-end developer - London

With two and a half years of professional experience as front-end developer, I am now looking for new opportunities.

I like to work with the latest JavaScript frameworks, such as React and Node. Meal Planner

Meal Planner

The meal planner has been created to help the user to plan what they are eating during the week. It is a project made with React calling three different APIs:

  • Firebase: saves and updates the data on the main board
  • Zomato: gets the list of restaurants near you
  • Edamam: gets recipes from an ingredient

Technologies used

  • React
  • APIs: Zomato, Firebase, Edamam
  • Responsive with Flexbox

Meal Planner

Details

The meal planner has been created to help the user to plan what they are eating during the week. It is a project made with React calling four different APIs:

  • Firebase: saves and updates the data on the main board
  • Zomato: gets the list of restaurants near you
  • Edamam: gets recipes from an ingredient
  • GraphHopper: calculates the distance and time between two locations

Technologies used

  • React
  • APIs: Zomato, Firebase, Edamam, GraphHopper
  • Responsive with Flexbox
